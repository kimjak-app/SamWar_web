# SamWar_web PROJECT BRIEF

## Project Identity

SamWar_web is the HTML MVP prototype for SamWar.

SamWar is a turn-based hero strategy game where ancient East Asian heroes compete to unify the Chinese mainland, the Korean peninsula, and the Japanese archipelago.

## Core Fun

Gain heroes, capture cities, expand your nation, and unify the world.

## Core Direction

- Turn-based strategy
- Hero-centered gameplay
- City occupation and expansion
- Data-driven structure
- HTML MVP first, engine migration later

## Reference Prototype

The existing Godot prototype is treated as the reference source for the early world map and basic battle loop.

## Development Principle

Follow Dream Team rules:

- Modular separation
- Data separation
- No monolithic file
- Clear project state tracking through the agent folder
