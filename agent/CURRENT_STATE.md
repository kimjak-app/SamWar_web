## Implemented Structure

- `css/main.css` contains the title screen placeholder styling.
- `js/main.js` initializes the app entry point.
- `js/core/app_state.js` owns initial app state assembly.
- `js/ui/layout_ui.js` renders the placeholder UI.
- `data/` contains placeholder factions, cities, heroes, and skills.
- `agent/HANDOFF_TO_CHATCOACH.md` records the short end-of-task handoff.
